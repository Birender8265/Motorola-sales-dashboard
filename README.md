📊 Motorola Mobile Sales Dashboard
📌 Project Overview

This project is an interactive Mobile Sales Analysis Dashboard built using mobile sales data. The goal of the project is to analyze sales performance, customer behavior, and transaction trends across different dimensions such as time, city, brand, and payment methods.

The dashboard provides meaningful insights that help in data-driven decision making for sales strategy and business growth.


🎯 Objectives
To design and develop an interactive Power BI dashboard that analyzes mobile sales data, uncovering trends in sales, customer behavior, and regional performance to enable informed business decisions.


🚀 Key Features

📅 Monthly Sales Trends

Visualizes total quantity sold across months

Identifies peak and low-performing periods

🌍 Sales by City

Geographical representation of sales distribution across cities in India

Highlights high-performing regions

📱 Sales by Mobile Model

Comparison of top-selling models (e.g., iPhone SE, Vivo Y51, OnePlus Nord)

🏷️ Brand Performance

Sales comparison between brands like Apple, Samsung, and OnePlus

💳 Payment Method Analysis

Breakdown of transactions via UPI, Debit Card, Credit Card, and Cash

⭐ Customer Ratings Analysis

Distribution of customer ratings (1–5 stars)

Helps assess customer satisfaction

📆 Day-wise Sales Trend

Tracks which days generate higher sales volume

🔍 Interactive Filters

Month-wise filtering

Brand and payment method slicers

🛠️ Tools & Technologies Used

Power BI – Data visualization & dashboard creation

Excel / CSV Dataset – Data source and preprocessing

DAX (Data Analysis Expressions) – Measures and calculations

Data Cleaning & Transformation – Power Query

📊 Key Metrics

Total Sales: 769M

Total Quantity Sold: 19K

Total Transactions: 4K

Average Sales Value: ~40K

💡 Insights Gained

Certain months (e.g., March, July) show peak sales trends

Metro cities contribute significantly to total sales

Balanced usage of payment methods, with slight dominance of UPI

High customer ratings indicate overall positive user satisfaction

Specific models consistently outperform others
